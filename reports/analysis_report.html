Compound Activity Predictor

Bench to Bytes portfolio analysis report. Uses only public or synthetic data.

Hypothesis

Interpretable molecular descriptors should explain a meaningful share of compound potency variance.

Data

Synthetic public-safe compound activity table inspired by ChEMBL and TDC bioactivity schemas; the RDKit adapter is included for real public SMILES workflows.

Process

  1. Generated compound descriptors, scaffolds, and pIC50 values with known descriptor signal.
  2. Fit a regularized linear regression baseline.
  3. Exported R2, RMSE, MAE, predictions, and residuals.
  4. Created a labelled predicted-versus-actual figure for model fit inspection.

Outputs

  • data/compound_activity.csv
  • outputs/regression_metrics.csv
  • outputs/predictions.csv
  • figures/predicted_vs_actual.svg

Interpretation

The descriptor baseline recovers moderate signal, showing that simple, chemically interpretable features can be useful before moving to fingerprints, graph models, or target-specific public ChEMBL data.
